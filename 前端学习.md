# 前端学习

## Html

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
h标签内容都是独占一行
<p>这是一个段落。</p>
p标签内容会自动换行
<hr> 一条横线
<strong>这是加粗的文本</strong>
<em>这是斜体的文本</em>
<ins>这是下划线的文本</ins>
<del>这是删除线的文本</del>
```

```txt
块级元素与内联元素：
  块级元素：
  段落（p）
  标题（h1~h6）
  列表（ul、ol、li）
  表格（table、tr、td）
  div


  内联元素：
  文字（strong、em、ins、del）
  链接（a）
  图片（img）
  换行（br）
  水平规则（hr）


  块级元素可以包含内联元素和其他块级元素。（特殊情况p和内联元素一样）
  内联元素只能包含内联元素。
```

```
内联标签img
img标签用于在HTML文档中嵌入图片。
  他所有的属性有：
  src：图片的URL地址
  alt：图片的替代文本
  width：图片的宽度
  height：图片的高度
  title：图片的标题


alt是加载不出来显示的内容
title是鼠标悬浮显示的内容
宽高只设置一个就行另一个等比例缩放，当然一般来说不这样设置，而是通过css来设置宽高
```

```
相对路径寻找图片
  1. 图片和html文件在同一目录下，使用./或者直接写
  2. 图片在html文件的上一级目录，使用../
  3. 图片在html文件的下一级目录，使用./目录名/或者直接写目录名/
```

```
视频和音频
有关video标签的属性
  1. src 视频地址
  2. autoplay 自动播放
  3. muted 静音
  4. loop 循环播放
  5. controls 显示播放控件
  6. width 视频宽度
  7. height 视频高度
  8. poster 视频海报，还没播放时显示的图片


  自动播放需要静音
  键和值一样只写键就行

  <video autoplay muted loop>
    <source src="../video/ultra_2_2.mp4" type="video/mp4">
    <h2>您的浏览器不支持 video 标签</h2>
  </video>
<audio>和<video>几乎一模一样


音频浏览器不允许自动播放
```

```
超链接

a标签属性介绍：
  1. href 超链接地址 可以放外部链接也可以放内部链接，外部链接可以是邮箱或者exe文件
  2. target 超链接打开方式
    _blank 新窗口打开
    _self 当前窗口打开
  3. title 超链接标题 鼠标悬浮显示文字

  锚点链接：
  就是从一个地方点击后到达另一个地方，这个就是使用#+id，然后在要到达的地方的id写为id就行。
  想要平滑过渡可以使用css
  html{
  scroll-behavior: smooth
  }
```

## CSS

选择器：

```
  基本选择器
  1. 标签选择器
    选择所有的标签元素
    语法： 标签名
  2. 类选择器
    选择所有的类元素
    语法： .类名
  3. ID选择器
    选择所有的ID元素
    语法： #ID名
  4. 通配符选择器
    选择所有的元素
    语法： *




  讲一下关系选择器
  1. 后代选择器
  2. 子选择器
  3. 相邻兄弟选择器
  4. 通用兄弟选择器
  语法举例：
  1. 后代选择器
    选择所有的后代元素
    语法： 父元素 后代元素
  2. 子选择器
    选择所有的子元素
    语法： 父元素 > 子元素
  3. 相邻兄弟选择器
    选择第一个相邻兄弟元素
    语法： 元素 + 元素
  4. 通用兄弟选择器
    选择所有的通用兄弟元素
    语法： 元素 ~ 元素
```

文本布局：

```
文本对齐：
text-align：left right center justify(两端对齐)   只对块级元素有效果

首行缩进：
text-indent: 2em首行缩进2

字间距：用于设置文本字符的间距
letter-spacing: npx

行高：
line-height: npx
行高等于盒子高度，可以实现单行文本垂直居中，多行文字确实没法了。
```

font简写：

```
语法：font: font-style font-weight font-size/line-height font-family;
其中 font-size 和 font-family 是必须写
其他可以省略，默认显示
```
